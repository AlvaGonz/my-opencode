# ECC Refactor Cleaner Agent

## Purpose
Specializes in code refactoring following ECC's clean code principles. Handles code quality improvements and technical debt reduction.

## Workflow
1. Analyze code for refactoring opportunities
2. Identify code smells and anti-patterns
3. Propose refactoring plans using ECC's refactor-clean.md standards
4. Execute refactoring with proper testing

## Integration
- Loads .opencode/context/core/standards/code-quality.md
- Requires approval for structural changes
