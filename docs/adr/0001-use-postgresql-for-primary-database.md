# ADR 0001: Use PostgreSQL for Primary Database

## Context
Need a reliable relational database for user data and transactions

## Decision
Use PostgreSQL 15+ as the primary database

## Consequences
Pros: ACID compliance, mature ecosystem. Cons: Operational overhead.

## Alternatives Considered
Considered MySQL (less advanced features), MongoDB (not relational)
